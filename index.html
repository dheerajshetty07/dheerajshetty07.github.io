H1B Compass

Realtime Voice Agent

Realtime Voice Agent is a browser-based AI receptionist built with LiveKit Cloud SDK, Google Gemini 3.1 Flash Live, and Vercel Frontend to answer questions and capture appointment bookings through natural real-time voice conversations.

Taste Trails

Taste Trails is a consumer web app for food and travel discovery, combining curated recommendations and a responsive browsing experience to help users explore places and plan outings more easily.

Daily AI News Digest

Daily AI News Digest is an n8n workflow that scans 17 AI RSS feeds, ranks stories, uses NVIDIA NIM to write a Telegram-ready summary, and logs each run to Google Sheets for faster daily monitoring.

WhatsApp Concierge

WhatsApp Concierge is an n8n assistant that uses WhatsApp Business Cloud, Google Calendar, Google Sheets, Gmail, and NVIDIA NIM to answer questions, book appointments, and escalate complex requests automatically.

Instagram Assistant

Instagram Assistant is an n8n DM workflow that combines the Instagram Graph API, Groq, and Google Sheets to classify intent, draft contextual replies, track lead status, and flag conversations for human follow-up.

Extract Text From Anything

Extract Text From Anything is an n8n document-intake workflow that uses Telegram or Google Drive triggers, NVIDIA NIM, Google Docs, and Google Sheets to turn mixed file types into structured, reusable text output.

Portfolio Statement Analyzer

Portfolio Statement Analyzer is a document workflow that uses Google Drive, LlamaParse, Groq, Airtable, and Gmail to extract balances, holdings, risk flags, and summaries from portfolio statements with far less manual review.

Job Hunt Automation

Job Hunt Automation is a multi-workflow n8n system that uses Adzuna, JSearch, RSS, Greenhouse, Groq, Google Drive, Google Sheets, Gmail, and LaTeX resume generation to find roles and streamline tailored applications.
